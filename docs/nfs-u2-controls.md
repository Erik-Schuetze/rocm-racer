# Need for Speed - Underground 2 - Controls for PS2

Left stick - steer
X or right stick up - accelerate
Square or right stick down - brake/reverse
Triangle - change camera
Circle - look back

R1 - e-brake
L1 - nitrous
R2 - shift up (manual transition only)
L2 - shift down (manual transition only)

D-pad up - enter shop/challenge opponent
D-pad down - career stats (career mode only)
D-pad right - display DMS (career only)
D-pad left - map (career only)

Select - reset car
Start - pause
